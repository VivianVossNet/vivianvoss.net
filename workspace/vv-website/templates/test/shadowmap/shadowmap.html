Shadow Map

Real-time building shadow projection on an interactive map. Given a position, date, and time, it calculates which buildings cast shadows where and determines whether each restaurant or café is currently in the sun or in the shade.

Stack
Mapbox GL JS v3.3, ~250 lines of vanilla JavaScript
Dependencies
None (beyond Mapbox CDN)
Requires
A free Mapbox access token
Size
12 KB (ZIP)

The archive contains two files:

  • shadow-map.html — working reference implementation. Insert your Mapbox token and open in a browser. No build step
  • shadow-map.llm.md — integration guide for AI coding assistants (Claude Code, Cursor, etc.). Documents every function, known pitfalls, and security best practices so the assistant can integrate the shadow logic into your project
